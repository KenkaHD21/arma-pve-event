# PVE Event – Arma Reforger Übersicht

Übersicht, Funkplan, Organigramm und LADEF für Arma Reforger PVE-Events (20–40 Spieler).

## Inhalte

- **Übersicht** – Event-Infos, Struktur, Rollen
- **Organigramm** – Befehlsstruktur (PL → Squads → Fire Teams)
- **Funkplan** – Frequenzen (Command, JTAC, Squads 1–4)
- **Slotliste** – Interaktive Slotliste mit Speicherfunktion
- **Mods** – ###### Mod-Liste
- **LADEF** – 5-Absatz-Befehl (Lage, Auftrag, Durchführung, Einsatz, Führung)
- **Checkliste** – Vor dem Event

## GitHub Pages einrichten

1. Repository auf GitHub erstellen
2. Dateien hochladen (oder per Git pushen)
3. **Settings** → **Pages** → **Source**: „Deploy from a branch“
4. **Branch**: `main` (oder `master`), **Folder**: `/ (root)`
5. Nach 1–2 Minuten: `https://deinusername.github.io/repo-name/`

## Lokal testen

Ordner öffnen und `index.html` im Browser laden – oder:

```bash
python -m http.server 8000
# Dann http://localhost:8000 öffnen
```
